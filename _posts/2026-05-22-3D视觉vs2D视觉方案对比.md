## 一、问题背景

"用2D还是3D？"这是工业视觉方案选型中最常见的纠结。

某新能源电池厂的技术负责人曾问我们："检测电芯极片表面缺陷，2D就够了吧？为什么供应商推荐3D？" 某汽车零部件厂的老板也很困惑："2D方案报价15万，3D方案报价48万，贵3倍真的值得吗？"

**结论先行**：大多数场景下，2D+AI已经足够；只有在高度/深度/立体结构检测是刚需时，才值得为3D多付那笔钱。

本文提供一套可复用的选型判断框架。

## 二、技术原理对比

| 维度 | 2D视觉 | 3D视觉 |
|------|--------|--------|
| **成像原理** | 单相机拍摄平面图像（灰度/彩色） | 激光三角测量/结构光/双目立体视觉/ToF |
| **输出数据** | 平面像素矩阵（X×Y） | 点云数据或深度图（X×Y×Z） |
| **分辨率精度** | 微米级（依赖镜头和像素） | 微米到亚微米级（依赖技术路线） |
| **处理速度** | 快（毫秒级，单帧处理） | 相对慢（百毫秒级，点云计算量大） |
| **环境抗干扰** | 对环境光敏感，需要稳定光源 | 激光方案对自然光抗干扰强；结构光对环境光敏感 |
| **硬件成本** | 低（几千到几万） | 高（几万到几十万） |
| **算法复杂度** | 低到中 | 中到高 |

## 三、应用场景适配矩阵

### 2D视觉擅长的场景

| 场景 | 典型应用 | 为什么2D就够了 |
|------|----------|---------------|
| 表面缺陷检测 | 划痕、污渍、色差、印刷缺陷 | 缺陷特征在平面图像上清晰可见 |
| OCR/字符识别 | 标签读取、二维码识别、激光刻字 | 字符信息存在于二维平面上 |
| 尺寸测量（平面） | 孔径、间距、轮廓尺寸 | 测量维度为平面X-Y方向 |
| 装配验证 | 零件有无、位置正确性 | 定性判断，不涉及高度 |
| 颜色检测 | 色差判定、颜色分类 | 色彩信息在二维图像中完整保留 |

**2D方案的2026年新增能力**：AI大模型加持后，2D视觉不再局限于传统规则。视觉大模型可以理解图像的语义信息——例如不仅识别"有划痕"，还能区分划痕深度等级、判断是否影响功能。这意味着2D方案的能力边界大幅扩展。

### 3D视觉擅长的场景

| 场景 | 典型应用 | 为什么必须用3D |
|------|----------|---------------|
| 高度/深度测量 | 台阶高度、焊点高度、凹陷深度 | 需要Z轴数据 |
| 体积/形状检测 | 铸件完整性、注塑件变形 | 二维图像无法判断立体形态 |
| 共面度/平面度检测 | PCB板翘曲、连接器引脚共面度 | 需要三维空间数据 |
| 机器人引导（复杂姿态） | 散乱抓取、异形件装配 | 需要物体的空间位姿信息 |
| 焊道质量检测 | 焊道宽度×高度×外形 | 焊道的三维形态是关键质量指标 |

## 四、成本分析：为什么3D贵这么多？

很多工厂决策者看到报价单就困惑：同样是视觉检测方案，为什么3D比2D贵3-5倍？拆解成本结构：

| 成本项 | 2D方案 | 3D方案 | 价格差异原因 |
|--------|--------|--------|--------------|
| 相机+镜头 | ¥3,000-15,000 | ¥15,000-80,000 | 3D相机（激光/结构光）硬件复杂度远高于2D |
| 光源 | ¥1,000-5,000 | ¥2,000-10,000 | 3D方案常需特殊激光或结构光光源 |
| 工控机 | ¥5,000-15,000 | ¥15,000-50,000 | 点云数据处理算力需求大 |
| 软件/算法授权 | ¥5,000-20,000 | ¥20,000-100,000 | 3D算法开发难度和周期远高于2D |
| 系统集成 | ¥5,000-15,000 | ¥10,000-40,000 | 3D标定和调试复杂度高 |
| **合计（单工位）** | **¥20,000-70,000** | **¥65,000-280,000** | — |

**但要注意**：3D方案虽然在单工位成本上更贵，但在某些场景下能省去多个2D工位的组合，总体拥有成本（TCO）反而更低。例如：一个需要同时检测高度、平面度和表面缺陷的零件，可能需要3个2D工位（正上方、侧面、斜视），而1个3D工位即可全覆盖。

## 五、选型决策框架

按以下4个问题逐层判断：

```
Q1: 检测项是否涉及高度/深度/立体形态？
  ├─ 是 → Q2
  └─ 否 → 2D方案 + AI大模型

Q2: 精度要求是否达到微米级（如 ±0.01mm）？
  ├─ 是 → 3D方案（激光三角测量路线）
  └─ 否 → Q3

Q3: 检测速度要求是否超过 60件/分钟？
  ├─ 是 → 2D+3D混合方案（关键工位3D，其余2D）
  └─ 否 → 3D方案（结构光或ToF路线即可）

Q4: 预算是否充足（单工位预算 > ¥15万）？
  ├─ 是 → 3D方案
  └─ 否 → 2D方案 + 精细化机械定位辅助
```

## 六、OVR一站式方案的建议

我们的经验是：**不要为了技术选技术，要从产线实际需求出发**。

OVR方案内置了2D+3D统一调度框架：
- 同一套软件平台可管理2D和3D工位
- 根据每道工序的实际需求灵活配置
- 2D和3D工位的数据在同一平台上汇聚分析

这意味着你不用纠结"整体选2D还是3D"——可以产线上该用2D的地方用2D，该用3D的地方用3D，一套系统统一管理。
